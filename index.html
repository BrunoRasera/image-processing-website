    Como usar

    Este site permite a aplicação de diversos efeitos de processamento de imagens, com o intuito de facilitar o entendimento e visualização de como eles funcionam.

    Primeiro passo:

    Clique no botão "Selecionar imagem". É possível selecionar imagens disponibilizadas pelo site ou fazer o upload de alguma imagem do próprio computador.

    Segundo passo:

    Clique em "Aplicar efeitos". Aqui é possível selecionar diversos efeitos que serão aplicados a imagem selecionada, e o resultado será mostrado do lado direito da imagem original.

Thresholding é um método de segmentação de imagens que divide uma imagem em duas regiões: fundo e objeto de interesse. Isso é feito através do estabelecimento de um limiar (threshold) que é comparado com cada pixel da imagem. Pixels com valores acima do limiar são marcados como objeto de interesse e pixels abaixo do limiar são marcados como fundo.

Otsu é um algoritmo de limiarização de imagens desenvolvido por Nobuyuki Otsu em 1979. Ele é usado para achar o limiar ideal que melhor separa o objeto de interesse de um fundo. O algoritmo funciona calcularando a variancia entre duas classes, fundo e objeto, em relação ao limiar. O limiar que maximiza essa variancia é escolhido como o limiar ideal. Esse algoritmo é considerado melhor do que o limiar simples, pois ele é capaz de determinar automaticamente o limiar ideal sem a necessidade de escolher manualmente o limiar.

Thresholding adaptativo é uma técnica de limiarização que ajusta dinamicamente o limiar para diferentes áreas da imagem. Em vez de usar um único limiar para toda a imagem, ele calcula diferentes limiares para diferentes regiões. Isso é útil quando a iluminação da imagem varia significativamente em diferentes áreas, o que pode fazer com que um limiar fixo não funcione bem.

O Canny Edge é um algoritmo de detecção de bordas desenvolvido por John F. Canny em 1986. É amplamente utilizado para detectar bordas em imagens, especialmente em aplicações de visão computacional e processamento de imagens. O algoritmo funciona detectando mudanças significativas na intensidade de cor de um pixel para o próximo, conhecido como gradiente. Ele usa uma série de etapas para remover ruído e manter apenas as bordas mais fortes e precisas. Essas etapas incluem suavização da imagem, cálculo do gradiente de intensidade, nivelamento de histerese e eliminação de bordas finas. O algoritmo é capaz de detectar bordas tanto internas quanto externas, e é considerado um dos métodos mais precisos e robustos para detecção de bordas.

A equalização de histograma é uma técnica de processamento de imagem que visa aumentar o contraste em uma imagem. Ela é frequentemente usada para melhorar a visualização de imagens com baixo contraste, como imagens de raio-x ou fotografias antigas. O objetivo é redistribuir os pixels de uma imagem de forma que a distribuição de intensidade de cor fique mais uniforme, de modo que todos os níveis de cor tenham aproximadamente a mesma quantidade de pixels.

Para melhor observar seus resultados, use a imagem do gato disponível na seleção de imagens.

A Transformada de Fourier é uma técnica matemática amplamente utilizada no processamento de imagens para analisar e filtrar a informação de frequência contida nas imagens.

Esta representação permite a análise de diferentes componentes de frequência presentes na imagem, como ruído, textura, bordas, etc. Além disso, também permite a aplicação de filtros para remover ou reforçar componentes de frequência específicos, como por exemplo, filtros passa-baixas e passa-altas. A Transformada de Fourier é amplamente utilizada para diversas aplicações, incluindo compressão de imagens, reconstrução de imagens, análise de frequência e filtragem.

A segmentação Watershed é um algoritmo de segmentação baseado em marcas, que é utilizado para segmentar imagens. Ele funciona criando uma grade de marcas sobre a imagem de entrada, onde cada marca representa uma região diferente. O algoritmo então propaga as marcas através da imagem, levando em conta características como intensidade de pixel, cor e distância, para criar regiões distintas. O objetivo final é dividir a imagem em regiões conexas, onde cada região representa uma parte diferente da imagem, como objetos ou fundo.

Para melhor observar seus resultados, use a imagem das moedas disponível na seleção de imagens.

Ruído Salt and Pepper é um tipo de ruído que ocorre em imagens digitais, que é caracterizado por pixels brancos ou pretos aleatoriamente espalhados pela imagem. Esse tipo de ruído é causado por erros de amostragem ou problemas de transmissão de dados. É chamado de Salt and Pepper devido ao fato de que os pixels afetados parecem ser pontos de sal e pimenta espalhados pela imagem.

Ruído de Poisson é um tipo de ruído estatístico que é comumente encontrado em imagens digitais adquiridas por meios de detecção de luz, como câmeras digitais e sensores de imagem. Ele é causado pelo processo de conversão de luz para corrente elétrica no sensor de imagem, que gera eventos aleatórios de contagem de fótons. Como resultado, o nível de ruído em cada pixel é proporcional à intensidade de luz medida nesse pixel. O ruído de Poisson é caracterizado por sua distribuição de probabilidade poissoniana, o que significa que é mais provável que os pixels com intensidades mais altas tenham mais ruído.

Em processamento de imagens, uma máscara, também conhecida como kernel, é uma matriz pequena de números que é aplicada a uma imagem para realizar operações de processamento de imagem, como filtragem, detecção de bordas e transformações morfológicas. Essas operações envolvem aplicar a máscara a cada pixel da imagem, multiplicando seus valores pelos valores correspondentes na máscara e, em seguida, somando esses valores para obter um novo valor para esse pixel.
